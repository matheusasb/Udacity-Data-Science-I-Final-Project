Introdução¶

No processo de anlise de dados do projeto Titanic foi utilizado um banco de dados em formato ".csv" provido por este desafio do Kaggle.

O Titanic foi um grande navio do comeo do sculo XX famoso por ter sido considerado como o mais luxuoso e mais seguro de sua poca, gerando lendas que diziam que eram impossvel de ser afundado. Fato que infelizmente a histria se comprovou ao contrrio ao naufragar colidindo com um iceberg e causando grande perda humana.

Metodologia¶

A metodologia utilizada nesse projeto foi a apresentada nesse curso sendo dividida nas seguintes etapas:

Etapa Descrio
1 Fazer perguntas - Hipteses
2 Limpar os dados
3 Realizar a anlise exploratria de dados
4 Tirar concluses
5 Comunicar os resultados

Alm disso, buscou-se ao mximo possvel utilizar storytelling para comunicar as informaes.

Objetivo:¶

Iremos realizar um estudo desses dados, e identificar quais so os fatores com suas respectivas variveis que levaram ou no, a uma maior influncia na mortalidade das pessoas envolvidas no acidente.

#Funo que pega os nomes das colunas passa para letra minuscula e separa espaco por "_"
def adjustby_lower_and_line(df):
    """"
    Funo que utiliza do mtodo .rename() do pandas para renomear as colunas de um dataframe
    Alm disso,  usado uma funo lambda combinada com os mtodos .strip(), .lower() e .replace()
    Para retirar os espaos das estremidades, colocar tudo em letra minscula e substituir os espaos
    internos para o "_".
    Argumentos: df = um dataframe
    Retorna: Um dataframe com os nomes das colunas alterados
    """
    new_names = df.rename(columns=lambda x: x.strip().lower().replace(" ", "_"))
    return new_names

df_titanic_clean = adjustby_lower_and_line(df_titanic_clean)
